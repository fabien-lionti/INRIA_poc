📋 Export — Rapport HCERES

Personnalisez la structure du rapport, exportez le prompt, puis demandez à Antigravity de rédiger le rapport complet.

📐 Structure du rapport HCERES

Cette structure définit les sections et sous-sections qu'Antigravity utilisera pour rédiger le rapport. Vous pouvez la modifier librement pour adapter le rapport à vos besoins.

🚀 Générer le prompt
🤖 Powered by Antigravity

Le rapport sera rédigé par Antigravity à partir des données du dashboard et de la structure que vous avez définie. Aucune clé API nécessaire.

💡 Comment ça marche
1
Personnalisez la structure

Modifiez la structure du rapport ci-dessus. Ajoutez, supprimez ou réorganisez les sections selon vos besoins.

2
Exportez le prompt

Cliquez sur le bouton pour télécharger un fichier .txt contenant toutes les données + la structure.

3
Demandez à Antigravity

Dans votre conversation, demandez « Écris le rapport HCERES » — Antigravity lit les données et rédige le rapport complet.

💡 Astuce : Antigravity a déjà accès à vos fichiers CSV. Vous pouvez simplement lui demander de rédiger le rapport sans exporter le prompt !
