# Beamer Presentation

Short presentation, designed for about 5 minutes, of the HAL ACENTAURI HCERES POC.

Compile:

```bash
cd presentation
pdflatex hal_acentauri_hceres_poc_presentation.tex
```

The source file contains 5 slides:

1. Context and need
2. Goal of the POC
3. Framing prompt given to the AI
4. Generated pipeline
5. Results, limitations, and next steps
